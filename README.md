# 🌿 MiintSwap

A beautiful, fast Solana token swap interface powered by Jupiter.

## Features

- 🚀 **Fast swaps** - Powered by Jupiter aggregator for best prices
- 💎 **Beautiful UI** - Modern dark theme with glassmorphism effects  
- 🔐 **Secure** - Non-custodial, your keys stay in your wallet
- 📱 **Responsive** - Works on desktop and mobile
- 🎯 **Simple** - Clean, intuitive interface

## Tech Stack

- **Next.js 14** - React framework
- **Tailwind CSS** - Styling
- **Solana Web3.js** - Blockchain interaction
- **Wallet Adapters** - Phantom, Solflare support
- **Jupiter API** - Swap aggregation

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Deploy to Vercel

```bash
vercel --prod
```

## Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_RPC_URL=https://api.mainnet-beta.solana.com
```
